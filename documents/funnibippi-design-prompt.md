# Funni Bippi — Design Prompt

## Overview

Design **Funni Bippi**, a chat-with-strangers web app with a friendly, playful, and social personality.
Think of it as the lovechild of Slack's structured layout and a bubble tea shop's energy — organized
but alive, clean but warm. The app should feel welcoming to someone who just landed on it for the
first time with zero context.

**Tagline:** *"World without strangers."*

---

## Visual Direction

- **Mood:** Friendly, fun, approachable — like texting a new friend, not a corporate tool
- **Color system:** A vibrant primary accent (think coral, teal, or sunny yellow — NOT purple
  gradients) with a soft neutral base. Support both **light mode** (warm whites, soft creams) and
  **dark mode** (deep charcoal, not pure black). User can also pick a theme color from a palette.
- **Typography:** Pair a rounded, characterful display font (e.g. Nunito, Quicksand, or Poppins)
  with a clean readable body font. Text should feel warm, never sterile.
- **Shapes & Details:** Rounded corners everywhere. Bubbly chat bubbles. Soft drop shadows. Subtle
  gradient accents on buttons and highlights. A slight playful tilt or bounce on interactive elements.
- **Illustrations / Icons:** Use friendly line icons or filled icon style (no sharp corporate icons).
  Consider a small mascot or illustrated empty states.

---

## Layout — 3-Column (Slack-inspired but simplified)

### Left Sidebar (narrow)
- Funni Bippi logo / mascot at top
- User avatar + online status indicator (colored dot)
- Navigation icons: Chat, Profile, Settings
- Theme color picker (small swatches)
- Dark / Light mode toggle

### Center Panel (main chat area)
- **Top bar:** Stranger's anonymous avatar + generated name (e.g. "BlueOtter42") + gender badge +
  connection status
- **Loading state:** "Finding a match..." animated — playful spinner or pulsing dots with fun copy
  like *"Looking for someone cool..."*
- **Chat message area:** Bubbles aligned left (stranger) and right (you). Timestamps on hover.
  Emoji reactions on hover.
- **Bottom composer bar:**
  - Emoji picker button
  - Image attach button (clip icon)
  - Text input (rounded pill shape)
  - Send button (paper plane or lightning bolt icon)

### Right Panel (collapsible)
- Stranger's anonymous profile card (gender + generated name)
- **"Next Stranger"** button and **"Report"** button
- Fun icebreaker prompts: *"Ask them: Would you rather..."*

---

## Key Screens to Design

1. **Landing / Home screen** — Big friendly hero: *"World without strangers."* + a glowing
   **"Start Chatting"** CTA button + gender filter toggle (Everyone / Male / Female) with pill-style
   selector
2. **Matchmaking / Loading screen** — Animated, playful waiting state with fun copy
3. **Active chat screen** — Full 3-column layout as described above
4. **Settings panel** — Theme color picker grid + dark/light toggle + notification preferences
5. **Mobile responsive view** — Single column, bottom nav bar

---

## Micro-interactions & Animations

- Smooth slide-in for new messages
- Bouncy entrance animation for the match-found moment (confetti or a "✨ You matched!" toast)
- Emoji picker with a pop-up spring animation
- Typing indicator with animated bouncing dots
- Hover states on bubbles reveal timestamp + emoji reaction bar
- Send button pulse on hover

---

## Brand Identity

- **App name:** Funni Bippi
- **Tagline:** *"World without strangers."*
- **Logo concept:** A speech bubble with a small "bloop" ripple effect — like a water droplet
  hitting a surface, playful and rounded
- **Mascot idea:** A tiny round blob character with big friendly eyes peeking from the corner of
  empty states

---

## Do NOT

- Use purple gradients on white — it's overdone
- Make it look like a dating app (no big profile photos, no swipe UI)
- Use harsh colors or aggressive contrast
- Make it feel corporate or cold
- Use sharp / angular design elements
